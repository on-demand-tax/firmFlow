# FirmFlow MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build FirmFlow MVP — marketing site + internal app for time tracking, approvals, expenses, Google Drive integration, and dashboard — per `docs/superpowers/specs/2026-06-20-firmflow-design.md`.

**Architecture:** Single Next.js App Router monolith with Route Groups `(marketing)` and `(app)`. MongoDB/Mongoose for data, NextAuth Google OAuth for auth, Service Account for Drive. TDD with Jest + mongodb-memory-server. Vertical slices S0→S5, each producing testable software.

**Tech Stack:** Next.js 14+ (App Router), TypeScript, Tailwind CSS, Shadcn UI, Mongoose, NextAuth.js v4, googleapis, Jest, React Testing Library, mongodb-memory-server

**Spec:** `docs/superpowers/specs/2026-06-20-firmflow-design.md`

- [ ] **Step 5: Create `middleware.ts` (project root)**

```typescript
import { NextResponse } from 'next/server';
import type { NextRequest } from 'next/server';
import { getToken } from 'next-auth/jwt';
import { isProtectedAppPath } from '@/lib/middleware-utils';

export async function middleware(req: NextRequest) {
  const { pathname } = req.nextUrl;
  if (isProtectedAppPath(pathname)) {
    const token = await getToken({ req, secret: process.env.NEXTAUTH_SECRET });
    if (!token) {
      const login = new URL('/login', req.url);
      login.searchParams.set('callbackUrl', pathname);
      return NextResponse.redirect(login);
    }
  }
  if (pathname === '/login') {
    const token = await getToken({ req, secret: process.env.NEXTAUTH_SECRET });
    if (token) return NextResponse.redirect(new URL('/app', req.url));
  }
  return NextResponse.next();
}

export const config = { matcher: ['/app/:path*', '/login'] };
```
